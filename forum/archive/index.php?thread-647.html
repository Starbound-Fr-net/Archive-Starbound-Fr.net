Salut Salut ^^

J'ai trouver la solution sans avoir a demander a son amis d'envoyer son dossier de jeu.

Il suffit dans dans le dossier du jeu:

Program Files>Steam>steamapps>common>Starbound

ou

Aller sur Steam>Clique droit sur Starbound>Propriétés>Fichiers Locaux et Parcourir les fichiers locaux

Puis ensuite vous avez juste a supprimer le fichier starbound.xml dans le dossier principal.
Sa a réglé le problème pour moi.
