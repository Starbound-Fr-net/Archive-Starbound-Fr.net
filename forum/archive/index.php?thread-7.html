Chers amis, afin de maintenir une ambiance conviviale sur le site, certaines règles doivent impérativement être observées par tous, ces règles peuvent être classées en catégories :


Les règles de savoir vivre :
- Toute insulte, manque de respect avéré envers un autre joueur sera passible d'un kick/ban (définitif ou non selon la gravité de l'acte).
- Évitez les hors-sujets. La section "Salon" est faite pour cela.
- La présentation des membres n'est pas obligatoire mais est très fortement recommandé. Elle vous permet de mieux vous intégrer à la communauté
- Une orthographe convenable est exigée, pensez à utiliser un correcteur orthographique si vous avec du mal avec la langue française.
- Toute pub sur le forum entrainera un seuil d'avertissement ainsi qu'un potentiel ban, les seules autorisées (après présentation convenable) sont les chaines YouTube ici et les serveurs de jeu ici.
- Postez dans la bonne section, une description est disponible sous chaque titre de section, elle est importante.

- Tout acte de harcellement et/ou de discrimination (y compris de sexisme) peut être passible de bannissement définitif.

Autre :
- Nous recruterons des modérateurs que lorsque cela sera nécessaire.

En vous inscrivant, vous certifiez avoir lu.
Et par conséquent, vous vous engagez à respecter ces règles
Elles peuvent êtres mises à jour à n'importe quel moment, nous vous préviendrons en ajoutant un message à cette discussion.


Merci de votre attention,
Le Staff.
