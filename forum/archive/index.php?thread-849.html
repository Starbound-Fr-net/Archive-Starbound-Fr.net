Chaque secteur a une seed qui lui est propre. Trouver donc le secteur dont vous voulez modifier la seed grâce a son id (je pense que c'est assez explicite pour vous laisser le trouver)

Par exemple , je veux modifier la seed du secteur X:
Code :
{
      "id" : "sectorx",
      "name" : "X Sector",
      "sectorSeed" : 427469981495,
      "prefix" : "X",
      "sectorSymbol" : "/celestial/sector/x.png",
      "hueShift" : 90,

      "worldParameters" : {
        "threatLevel" : [5, 10],
        "unlockedBiomes" : [ "arid", "desert", "forest", "snow", "moon", "jungle", "grasslands", "magma", "tentacles", "tundra", "volcanic", "barren", "asteroidfield" ]
      }

id: le "nom" donné au secteur uniquement pour la programmation(Par exemple, on le trouve dans le fichier de l'upgrade pour vaisseau)
name: nom tel qu'il est affiché en jeu.
sectorSeed: on verra en dessou, mais pour faire clair c'est sur quoi le jeu se basera pour générer l'univers.
prefix: le nom qui apparaît dans celui des planètes.
sectorSymbol: la texture de l'icone tel que vue dans le navigateur.
hueShift: l'emplacement du bouton du secteur dans le navigateur.
worldParameters: les paramètres des planètes du secteur:
-threatLevel: Les niveaux des planètes,ici 5 est le minimum et 10 le maximum.
-unlockedBiomes: les biomes trouvables dans ce secteur (Notez qu'il n'y a pas tout les biomes disponibles, il manque, par exemple, le biome "ocean" ou encore le biome "arctic")


Donc, pour changer la seed, il faut remplacer le nombre de sectorSeed PAR UN NOMBRE DE 12 CHIFRES, PAS PLUS, NI MOIN. (J'ai pas testé, mais je ne voudrais pas être responsable du crash de votre jeu). Pensez a vérifier qu'il y a bien un virgule après, ça joue aussi pour éviter le crash.

Après , faites la manip dite plus haut et lancer le jeu.
Vous avez un terrain de jeu unique!
