Le fichier du vaisseau est propre au personnage. Le jeu se base juste des fichier qui sont dans ./assets/ juste pour le crée en début de partie, et ensuite pour récupéré la texture du fond.

Donc, si c'est un avec une image de fond personalisé, OUI, il faut qu'il l'est. Dans le cas contraire (il n'y a pas de fond OU le fond existe déja dans les dossiers (comme le station spatiale)), il n'en a pas besoin.

Si tu d'autres questions a propos de ça, n'hésite pas.
