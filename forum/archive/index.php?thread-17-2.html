Bon, j'ai fait deux MyCode pour les Spoilers.

une balise spoiler dite de noob
Code :
[spoiler]Je suis un message caché[/spoiler]
Alerte Spoiler de Noob:
Je suis un message caché

Un balise spoiler dite nominative : c'est à dire que l'on peu modifier le message d’entête.
Code :
[spoiler=Je suis le message d’entête]Je suis le message caché par les bourrelets de ta maman[/spoiler]
Spoiler : Je suis le message d’entête
Je suis le message caché par les bourrelets de ta maman
