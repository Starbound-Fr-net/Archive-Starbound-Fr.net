Un petit post aujourd'hui pour vous présenter un site que j'ai moi-même découvert dans l'après-midi, j'ai nommé Web of Trust.
Le principe du site est simple, vous pouvez donner votre avis sur n'importe quel site web qui vous tiens à cœur, ou même au contraire, dont la seule vue vous dégoute.
Le concept est assez simple, vous pouvez évaluer, avec une note entre 1 et 100, un site sur quatre critères :
- Crédibilité
- Fiabilité commerciale (Si c'est un site de vente, cela va de soit)
- Confidentialité/Respect de la vie privée
- Sécurité des mineurs (En gros, absence de contenu pouvant choquer)

De même, vous pouvez laisser un commentaire détaillant votre expérience sur tel ou tel site, pourquoi vous le conseilleriez ou pourquoi vous ne le conseilleriez pas.

Je vous propose donc de laisser votre avis, qu'il soit positif, négatif ou neutre (Peu importe, le but est de donner une image du site tel qu'il est vraiment !) pour le site sur lequel vous vous trouvez actuellement ! (J'ai trouvé aucun autre moyen de tourner ma phrase :'()
Voici donc l'adresse : http://www.mywot.com/en/scorecard/starbound-fr.net
Mais n'hésitez pas à aller voir la page d'autres sites !

Et sur ce je vous souhaite une bonne soirée/journée/matinée, selon l'heure à laquelle vous me lisez

EDIT : Et tant que j'y pense, n'oubliez pas de noter les commentaires, selon si vous les trouvez juste et cohérents, ou bien faux et injustifiés
EDIT 2 : WOT est aussi un plugin pour navigateur, très discret, il se présente sous la forme d'un rond vert, jaune ou rouge en haut de la page, qui vous indique si le site en question mérite votre confiance ou non. Un clic sur ce rond vous permet d'évaluer rapidement le site. Le même cercle apparaît aussi à côté de chaque lien dans votre moteur de recherche
