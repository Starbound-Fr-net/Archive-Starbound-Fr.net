Primo il te manque un fichier .frames avec ton .miningtool, donc ta drill ne recevra aucune animation.

Ensuite j'essaie de trouver l'erreur, j'y bosse et j'éditerais ça.

Edit : Oui alors voilà, j'suis aveugle.

Alors !

Les chemins sont bons, les fichiers semblent bons.

Mais il te faudra un fichier .modinfo à mettre à la racine de ton mode, de cette forme :

Code :
{
  "name" : "Dizenko's Drill",
  "version" : "Beta v. Furious Koala",
  "path" : ".",
  "dependencies" : [ ]
}

C'est le code minimal pour le jeu, y'a moyen de faire mieux avec Auteur et tout pour les Mod Manager, mais osef.

Il te faudra aussi un "player.config" à la racine de ton mod. Voilà à quoi il devrait ressembler :

Code :
{
  "__merge" : [],
  "defaultBlueprints" : {
    "__merge" : [],
    
    "tier1" : [     
      { "item" : "dizenko'sdrill" }  ]
  }
}

J'admets ici que ton item est dispo dès le tier 1, autrement dit dès le début. Si tu mets Tier2 il s'unlockera dès que tu auras la Metalwork Station je crois. Puisque tu dois le craft dessus, tu peux faire ça. Après puisqu'il se fait en Impervium, tu peux mettre des Tiers plus haut, mais débrouille toi pour voir lequel correspond à quoi.

Voilà, bisou. <3
