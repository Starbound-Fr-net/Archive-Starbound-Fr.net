Salut,

Pour accéder à notre IRC est hébergé chez EpiKnet, il faut soit passer par le lien en haut du site, soit avoir un client IRC (XChat 2, mIRC ou Pidgin par exemple).
HexChat, basé sur XChat, gratuit, est conseillé.
Pour vous connecter depuis un téléphone portable Android, je conseille AndroIRC.


Vous devrez utiliser les données suivantes si vous choisissez la seconde option (si lightIRC plante ou si vous préférez utiliser un logiciel) :
Serveur : eu.chat4all.org
Channel : #starbound-fr
Jeu de caractères : UTF-8 (très important pour la prise en charge des accents )
Votre pseudonyme
Votre Mot de passe (si vous avez enregistré votre pseudo)

La version pour navigateur internet est disponible ici, celui-ci utilisant lightIRC.
(Si celle-ci ne fonctionne pas, rafraichissez la page et/ou videz le cache de votre navigateur ; Autrement utilisez ceci )

Règles importantes :

Prendre le pseudo correspondant à celui sur le site.
Il est conseillé d'enregistrer votre pseudonyme pour éviter toute appropriation de celui-ci par une autre personne.
Un langage clair et lisible est recommandé.
Vous pouvez faire de la publicité de manière non-abusive, mais évitez d'entrer dans le salon, de balancer votre texte et de vous éclipser, ce ne sera pas bon pour votre réputation...
Respectez les autres et leur personnalité, ne soyez pas insultant ou trop condescendant ^^
Pas de racisme, nazisme, pédophilie ou affiliés, de près ou de (très) loin.
Ne pas faire de gags (ou être insultants) envers les personnes d'autres channels IRC.

Aide : Mode d'emploi de l'IRC,merci à Biscotteman.
