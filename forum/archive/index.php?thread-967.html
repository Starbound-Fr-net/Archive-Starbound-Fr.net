Bonjour à tous, aujourd'hui nous allons apprendre a créer un vaisseau, et a tout configurer pour qu'il fonctionne.

Dans ce tutoriel, je vais utiliser des fichiers déjà présent en jeu mais inutilisé par celui ci.

Premièrement :

Vous devez créer un nouveau dossier dans "mods" portant le nom de votre nouveau vaisseau ou un autre nom au choix.

Ensuite, vous allez avoir besoin de créer :

Citation :_Un dossier "Ships" contenant un autre dossier avec le nom de la race auquel appartiendra le vaisseau. Pour ce tutoriel nous allons utiliser le dossier "human".

*À placer dans le dossier ../mods/nomdumod/
_Un fichier nommé "universe_server.config"
_Un fichier nommé "nomdumod.modinfo"
*À placer dans le dossier ../mods/nomdumod/ships/human/
_Un fichier nommé "nomduvaisseau.structure"
_Deux fichier image en PNG (je reviendrais plus tard sur ces deux fichiers) qui doivent se nommé "nomduvaisseau.png" et "nomduvaisseaublocks.png"

Avant de partir sur la création graphique de votre vaisseau, nous allons nous attarder sur sa configuration :

ETAPE I
Alerte Spoiler de Noob:

Citation :Ouvrez le fichier "ship.modinfo" ce fichier va nous servir a déclaré au jeu qu'il y a une modification prioritaire a effectuer.

Code PHP :
{
"name" : "",
"version" : "",
"path" : "."
} 
Pour plus d'information pour savoir compléter ce fichier, je vous invite a suivre "Le Guide Du Moddeur" mis en place par notre amis "Maxigregrze"

ETAPE II
Alerte Spoiler de Noob:

Citation :Ouvrez le fichier "universe_server.config"

Collez-y ce bout de code que nous allons décortiquer par la suite.
Code PHP :
{
  "__merge" : [],
  "speciesShips" : {
    "nomdelarace" : "/ships/human/nomduvaisseau.structure"
  },
} 

"__merge" : permet de modifier un fichier du jeu, sans avoir a réecrire totalement tout le fichier d'origine.
"speciesShips" : indique au jeu que nous allons modifier les vaisseau de départ d'une race
"nomdelarace" : à remplacer par le nom de la race choisit, peut servire aussi pour les race custom, le chemin d’accès ci-après, lui indique le fichier .structure qui sera utilisé comme vaisseau par le personnage.

Avec ça, vous êtes déjà en mesure de modifier le vaisseau de départ de votre race préféré.

Alerte Spoiler de Noob:
