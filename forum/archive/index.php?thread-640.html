Vous voulez joué du ACDC ou Nyan cat dans Starbound ? C'est très simple et en seulement trois étapes :

1) Trouver sur internet la partition .abc de votre musique et copiez la dans un fichier texte ( Grosse liste de partition : http://reissendewoelfe.de/abc/ )
2) Enregistrez ce dossier sur le bureau et modifiez l'extension ".txt" par ".abc"

3) Deplaçez votre partition ".abc" dans votre dossier "Songs" de Starbound
( ...\Steam\SteamApps\...\Starbound\assets\songs )

Voila ! plus qu'a choisir la partition dans le menu de l'instrument !
