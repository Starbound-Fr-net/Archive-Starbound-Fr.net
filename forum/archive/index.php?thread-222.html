J'viens de trouver une info que Tiy à révélé à propos des capacités qu'attribuent les armures des différents races ( comme d'hab, tout est susceptible de changer ). Je vais poste le screen de l'IRC, trouvé sur le reddit de Starbound ( merci Prrowlesis ) :



Traduction :
Citation :- Tiy : Vous voulez que je vous dise quelles sont les capacités spéciales des armures de chaque races ?
- Tiy : Alors pour l'instant,
- Tiy : Ca pourrait changer ...
- Tiy : Alors :
- Tiy : Les armures Avians vous permettront de planer.
- Tiy : Les armures Apex vous permettront de sauter plus haut, peut être courir plus vite.
- Tiy : Les armures Humaines vous permettront de porter plus d'objets. Cela changera peut être car ce n'est pas incroyable, quoi que très utile.
- Tiy : Les armures des robots vous permettront de miner plus rapidement. ( le nom de la race n'a pas encore été décidé )
- Tiy : Les armures Florans convertissent l'énergie solaire en énergie supplémentaire.
- Tiy : Les armures Hylotl vous permettront de respirer sous l'eau et de nager plus rapidement.
- Tiy : Vous obtiendrez ce bonus surement si le set est complet,
- Tiy : c'est l'armure qui vous procure ces capacités, pas la race.
- Tiy : ainsi, vous pourrez fabriquer des armures d'autres races,
- Tiy : les acheter ou les obtenir par échange.
- Tiy : Mais au début du jeu, vous ne pourrez que fabriquer les armures de votre propre race.
