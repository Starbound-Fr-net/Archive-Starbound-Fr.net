Bonjour,

Si comme moi les particules utilisées pour la pluie et la neige vous posent des problèmes de performances (il est vrai que cela bouffe pas mal notamment dans la phase du jeu bêta non optimisée), il est possible de désactiver tout cela sans poser de problème particulier au jeu.

Pour ce faire :

- Naviguez dans le répertoire assets\weather du répertoire de Starbound.
- Naviguez dans le répertoire rain
- Editez les deux fichiers de configuration présents comme ceci :

1. Remplacer la valeur density par 0
2. Remplacer dans chaque bloc présent la valeur timeToLive par 0

Vous pouvez faire pareil pour la neige (répertoire snow)

Et là magique, vous n'aurez plus la pluie qui viendra vous embêter, en revanche les éclairs par temps de pluie seront toujours présents.

Pensez à faire avant toute modification une sauvegarde des fichiers originaux.

Si cela peut vous aider à gagner quelques Fps, qui sont pour certains rares ces temps-ci alors c'est bien :)
