Salut je vous présente mon premier mod :p
La powered Pickaxe !

Elle est plus rapide qu'une pioche en diamant et mine en 4X4

La recette :
-Moltencore x1
-Diamant x8
-Pioche en diamand x1
-Pixel x2000

Pour installer le mod c'est très simple j'ai crée une console Windows qui automatise le tout vous pouvez vérifier le code c'est pas malveillant

Sous Windows:
Pour installer : PoweredPickaxe_Install
Pour le désinstaller une autre console est présente elle se nomme : PoweredPickaxe_Clean

Pour MAC ou Linux:
Glisser déposer le Dossier assets dans le zip
