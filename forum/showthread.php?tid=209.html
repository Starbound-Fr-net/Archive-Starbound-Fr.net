Bon je propose de faire ce topic pour faire une sorte de time-line réunissant les nouvelles du jeux qui sont publié un peu partout à petite dose pour aider les rédacteurs à avoir le contenu rassemblé pour faire des articles.

Je vais emmètre quelques règles basiques pour améliorer la lisibilité du Topic.
-les seuls post autorisé sont un post apportant une ou plusieurs nouvelles (comme un tweet concernant Starbound, un post d'un dev donnant un renseignement supplémentaire sur le jeux ou tout autre renseignement)
-Un post par nouvelles ou renseignement
-Evitez les doublons.
-Les LiveStream peuvent - être signalé, vous pouvez indiquer grâce à des mots son contenu mais vous ne devez pas linker vers une vidéo "pirate"(dans le sens ou ce n'est pas le souhait des développeurs), disponible sur Youtube par exemple, par respect envers les developpeurs.
EDIT : Vous pouvez finalement linkez sur le forum (discrètement) les vidéo diffusé sur youtube ou autres des Livestreams autours de starbound mais les rédacteurs je ne vous conseil pas de es mettre dans vos articles si vous ne souhaitez pas subir le courroux de http://starbound-fr.net/a-propos-des-videos/
-Et surtout : Interdiction formelles de faire des commentaires sur ce topic concernant les news ! Ce Topic doit seulement comporter des post apportant des nouvelles du jeux, Pour les commentaires ça ce passe ici : http://starbound-fr.net/forum/showthread.php?tid=210.

-Les fan-arts mis en avant par les développeurs (via un tweet par exemples sont autorisé)

RAPPEL : les commentaires ça ce passe ici : http://starbound-fr.net/forum/showthread.php?tid=210.
Merci
