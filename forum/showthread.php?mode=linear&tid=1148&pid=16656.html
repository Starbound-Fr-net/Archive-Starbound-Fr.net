Jeux : Bombe party

/!\ il faut se connecter avec un compte (au choix) Steam/Twitch/Twitter/Facebook/Google+

Règle : trouvr un mot contenant xx lettres avant que la bombe explose. On a 3 vies, le dernier survivant gagne la partie. ( une personne peut rejoindre sans forcement jouer)

Comment jouer : allez sur le lien si dessous puis rejoigner un serveur ( une fois join, il y a un bouton pour joindre la partie, si vous ne cliquer pas vous serer un "spectateur".
(N'importe qui peut en creer un (ici se sera un modo) puis vous pouvez modifier des regle que je vais vous montrer si dessous)

Règle modifiable : Dictionnaire = anglais/francais; durée minimum de la bombe : 0/5 (5 conseiller); le message d'aceuil ; acces visiteur : Tchat + jouer / Jouer / rien (jouer conseiller pour eviter qu'il s'entraide); et la possibiliter d'un live stream.

que peut faire la personne qui a creer le "salon" : il peut banir , creer de nouveaux modo (qui on les meme regle que lui), il peut choisir les règle.

se que peut faire les "visiteur" : jouer en cliquand sur "rejoindre la partie", tchatez sur le tchat si il est acctiver.

Lien : http://bombparty.sparklinlabs.com/
